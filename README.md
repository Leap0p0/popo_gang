## Configuration
Features :

**FR :** 
* Création de gang
* Création d'un point boss
* Création d'un coffre
* Suppression de gang avec reboot
* Suppression de gang via un nom !
* La langue (FR/EN)

**EN :**
* Gang creation
* Creation of a point leader
* Creation of a chest
* Removing gangs with reboot
* Gang removal via name!
* Language (FR/EN)
